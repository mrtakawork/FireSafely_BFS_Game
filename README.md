# 最遠點猜測遊戲

一個基於 React 的網頁遊戲，目標是在網格空間中找出距離起點最遠的點（考慮障礙物）。

## 遊戲規則

1. 遊戲會隨機生成多個起點（2-4個，根據網格大小）
2. 你需要猜測距離最近起點最遠的格子位置
3. 網格中包含障礙物（牆壁），需要繞過它們
4. 每次猜測會顯示該點到最近起點的最短路徑距離
5. 你有 5 次機會找到最遠點
6. 找到最遠點即可獲勝並獲得得分
7. 遊戲結束後會顯示所有格子的距離

## 功能特色

- 🎮 互動式網格遊戲板
- 🚧 **障礙物系統**：隨機生成的牆壁增加遊戲挑戰性
- 🎯 **多個起點**：每次遊戲有 2-4 個起點，需要考慮所有起點的位置
- 📊 實時顯示猜測點的距離（考慮障礙物的最短路徑）
- 🔢 **自定義網格大小**：支持 3x3 到 30x30 的自定義大小
- ⚡ 快速選擇按鈕：一鍵選擇常用網格大小（5x5, 8x8, 10x10, 12x12, 15x15, 20x20）
- 🏆 得分系統
- 📱 響應式設計，支援手機和桌面
- 🎨 現代化 UI 設計，美觀的漸變和動畫效果

## 遊戲玩法

1. 選擇或輸入網格大小（3-30）
2. 點擊「新遊戲」開始
3. 觀察起點位置（紫色格子，可能有多個）
4. 注意黑色格子是障礙物（牆壁），無法通過
5. 點擊你認為距離最近起點最遠的格子
6. 查看顯示的距離數值作為提示（這是繞過障礙物的最短路徑距離）
7. 在 5 次機會內找到最遠點即可獲勝！
8. 遊戲結束後可以查看所有格子的距離

## 遊戲機制

### 距離計算
- 使用 BFS（廣度優先搜索）算法計算最短路徑
- 距離是考慮障礙物後的實際路徑步數
- 每個格子到起點的距離是到**最近起點**的距離
- 最遠點是到最近起點距離最遠的可達點

### 障礙物
- 約 15% 的格子會成為障礙物
- 障礙物不會與起點重疊
- 如果某個點被障礙物完全阻擋，距離顯示為 ∞

### 起點
- 根據網格大小自動生成 2-4 個起點
- 起點位置隨機且不重疊
